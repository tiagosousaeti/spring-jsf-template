# Spring JavaServer Faces Template 
Projeto Java 8 com Spring e JSF

### Dependências
- Java 8
- Javax Face 2.2.15
- Jetty Maven 9.2.11
- Maven Compiler 3.2
- Servlet API 3.1.0
- Spring Core 5.0.2

### Executando o projeto
Requer o Maven configurado na sua máquina. No terminal, use: 
mvn jetty:run

Acesse:
http://localhost:8080/template-spring-jsf/
